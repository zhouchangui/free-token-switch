## 🌟 Core Concept

**TokensBuddy** aims to break the silo of API tokens. Do you have excess API credits or are you looking for cost-effective tokens?

- **🤝 Find a Buddy**: Share your spare tokens with friends to accelerate development together.
- **💰 Monetize Excess**: Exchange your unused tokens for USTC or JD Clawtip (in China) via the tokensbuddy.com platform.
- **🔌 Deep Integration**: Seamlessly switch and manage Claude Code, Codex, Gemini CLI, OpenCode, and OpenClaw.

## 🚀 Quick Start

### 1. Download & Install
Go to [Releases](https://github.com/farion1231/tokens-buddy/releases) to download the installer for your platform.

### 2. Configure Tokens
Add your API keys in the app; multiple provider presets are supported.

### 3. Share & Monetize
Visit [tokensbuddy.com](https://tokensbuddy.com) for the latest sharing codes or to join the monetization program.

## 🛠️ Technical Architecture

- **Frontend**: React + Tailwind CSS + Lucide Icons
- **Backend**: Tauri 2 (Rust)
- **Database**: SQLite
- **Deep Link**: Support `tokensbuddy://` for easy importing
