## 🌟 核心理念

**TokensBuddy (Token 搭子)** 旨在打破 API Token 的孤岛。你是否有很多用不完的 API 额度？或者你正在寻找高性价比的 Token？

- **🤝 寻找搭子**：一键将你多余的 Token 分享给好友，共同加速开发。
- **💰 闲置变现**：通过 tokensbuddy.com 平台，将你的闲置 Token 兑换为 USTC 或国内京东 Clawtip 额度。
- **🔌 深度集成**：完美支持 Claude Code、Codex、Gemini CLI、OpenCode 和 OpenClaw 的一键切换与管理。

## 🚀 快速开始

### 1. 下载安装
前往 [Releases](https://github.com/zhouchangui/tokens-buddy/releases) 下载对应平台的安装包。

### 2. 配置 Token
在应用中添加你的 API 密钥，支持多种 Provider 预设。

### 3. 分享与变现
访问 [tokensbuddy.com](https://tokensbuddy.com) 获取最新的分享码或参与变现计划。

## 🛠️ 技术架构

- **前端**: React + Tailwind CSS + Lucide Icons
- **后端**: Tauri 2 (Rust)
- **数据库**: SQLite
- **协议**: 支持 `tokensbuddy://` 深度链接导入
