## 🌟 核心理念

**TokensBuddy (Token 搭子)** は、API トークンの孤島を打破することを目指しています。余った API クレジットはありませんか？それともコストパフォーマンスの高いトークンをお探しですか？

- **🤝 仲間を探す**：余ったトークンを友人と共有し、開発を加速させましょう。
- **💰 収益化**：tokensbuddy.com プラットフォームを通じて、余ったトークンを USTC や京東 Clawtip に交換できます。
- **🔌 高度な統合**：Claude Code、Codex、Gemini CLI、OpenCode、OpenClaw のシームレスな切り替えと管理をサポート。

## 🚀 クイックスタート

### 1. ダウンロードとインストール
[Releases](https://github.com/farion1231/tokens-buddy/releases) から各プラットフォーム用のインストーラーをダウンロードしてください。

### 2. トークンの設定
アプリ内で API キーを追加します。多くのプロバイダープリセットをサポートしています。

### 3. 共有と収益化
[tokensbuddy.com](https://tokensbuddy.com) にアクセスして、最新の共有コードを取得したり、収益化プログラムに参加したりしてください。

## 🛠️ 技術スタック

- **フロントエンド**: React + Tailwind CSS + Lucide Icons
- **バックエンド**: Tauri 2 (Rust)
- **データベース**: SQLite
- **ディープリンク**: `tokensbuddy://` によるインポートに対応
